# application_metadata_applicants: ETH Board Open Research Data (ORD) Program Project Metadata and Report Data

ETH Open Research Data dataset, A colletion of metadata and report data
of a CHF 15 million ETH domain program dedicated to increase the
visibility and the impact of their research within the scientific
community, the economy and society as a whole.

## Usage

``` r
application_metadata_applicants
```

## Format

A data frame with 113 rows and 7 variables:

- project_id:

  Unique identifier for each project within the ETH Open Research Data
  dataset. (Examples: CONTRIBUTE/SOLARFUELSDB+, EXPLORE/OPENNEXUS-E,
  EXPLORE/FRAME)

- index:

  Sequential identifier for each entry in the dataset. (Examples: 3, 1,
  6)

- eth_domain_institution:

  Identifier for the institution within the ETH domain associated with
  the research project. (Examples: WSL, EPFL, PSI)

- function_title:

  Job title or position held by an individual associated with the
  research project. (Examples: Postdoctoral researcher, Deputy Director
  WASH R&D Center, Scientific collaborator)

- first_name:

  Given name of an individual associated with the research project.
  (Examples: Peter, Zhanyun, Mati Ullah)

- surname:

  Family name of an individual associated with the research project.
  (Examples: Schaffner, De Moraes, Feulner)

- postcode:

  Postal code associated with the address of an individual or
  institution. (Range: 0 to 9014)
